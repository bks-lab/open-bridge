# Bridge Deck — Visualization for The Bridge (coming soon)

**What it is:** an optional, **read-only** pixel-art rendering of your
Bridge as a literal spaceship bridge — agents appear as walking sprites
at their stations, with calendar and mandants tabs. It only reads Bridge
state and never writes to it.

**Status:** the `bridge-deck` repo is still **private and in
development**. It will be published when it ships; this guide will then
be expanded with setup and configuration details.

A config template for a standard Bridge layout already ships here:
[`examples/bridge-deck.config.yaml.template`](../examples/bridge-deck.config.yaml.template).

For now, use `/bridge-dashboard` or `/briefing` to get an overview of
your Bridge.
